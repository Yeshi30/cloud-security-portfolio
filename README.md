> Hands-on AWS security labs covering IAM, log monitoring, threat detection, and incident response.

## What It Covers

- Audit AWS environments using IAM least-privilege policies and MFA enforcement
- Monitor logs and trigger alerts with CloudWatch and SNS
- Automate incident response using EventBridge and Lambda
- Protect web applications with AWS WAF rules
- Detect threats using GuardDuty and Security Hub
